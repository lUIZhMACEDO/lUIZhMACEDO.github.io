# Luiz Macedo - Portfolio Website

A modern, data-driven portfolio website showcasing my projects, skills, and experience in Data Science and Engineering.

## 🚀 Features

- **Animated Data Visualizations**: Interactive charts using Chart.js
- **Particle Background Effects**: Dynamic particle system across all pages
- **Responsive Design**: Fully responsive and mobile-friendly
- **Smooth Animations**: CSS animations and transitions throughout
- **Project Showcase**: Detailed project pages with lightbox image viewing
- **Data Analytics Focus**: Themed around data science and analytics

## 📁 Project Structure

```
├── index.html              # Homepage with data visualizations
├── about.html              # About page with skills
├── projects.html           # Projects listing page
├── injury-analytics.html   # Premier League Injury Analytics project
├── DBproject.html          # Premier League Database project
├── soccerproject.html      # Soccer Analytics Site project
├── contact.html            # Contact page
├── styles.css              # Main stylesheet
├── script.js               # Main JavaScript file
├── images/                 # Image assets
├── projects/               # Project-specific files
└── DBproject/              # Database project images
```

## 🚀 Deployment

This site is ready for GitHub Pages deployment.

### To deploy on GitHub Pages:

1. Push this repository to GitHub
2. Go to repository Settings → Pages
3. Select the main branch as the source
4. Your site will be available at `https://yourusername.github.io/repository-name`

If your repository is named `username.github.io`, it will automatically be available at `https://username.github.io`

## 📝 Notes

- All file paths are relative, making it compatible with GitHub Pages
- External CDN resources (Chart.js, Font Awesome) are used for reliability
- The site uses modern CSS features and JavaScript ES6+
